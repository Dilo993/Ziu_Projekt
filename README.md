# Zaawansowany Interfejs Użytkownika - Projekt Laboratoryjny

## Opis projektu
Aplikacja stanowi nowoczesny panel administracyjny (Dashboard Layout) oparty na bibliotece React. Projekt integruje zaawansowane zarządzanie stanem asynchronicznym, mikrointerakcje, pełną dostępność WCAG oraz analitykę webową privacy-first zgodną z regulacjami RODO. Głównymi modułami systemu są trzykrokowy formularz rejestracji użytkownika z systemem detekcji porzuceń, dynamiczny eksplorator danych z zewnętrznego API oraz interaktywna lista zadań.

## Użyte technologie
- **Rdzeń aplikacji:** React 18, Vite, TypeScript
- **Biblioteka komponentów UI:** Material UI (MUI v6)
- **Zarządzanie routingiem:** React Router DOM v6
- **Zarządzanie stanem i warstwa danych:** TanStack Query v5 (React Query)
- **Silnik animacji:** Framer Motion & CSS keyframes
